Lập trình website chuyên nghiệp với ReactJS

Mục tiêu học tập:
- Hiểu rõ cách hoạt động của React, và vì sao React trở thành thư viện Frontend phổ biến nhất trong phát triển web.
- Hiểu rõ cách thức hoạt động và cách sử dụng các thành phần quan trọng nhất của React như: components, state, props, life cycle, events, hook, hocs, router
- Nắm vững cách quản lý dữ liệu trong ứng dụng
- Sử dụng Restful API để xây dựng web động
- Tối ưu hóa hiệu suất ứng dụng và xử lý vấn đề về bảo mật.
- Triển khai dự án Reactjs ra thị trường
